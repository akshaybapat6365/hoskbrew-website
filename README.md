# Hoskbrew - Physical Game Manufacturing for Indie Devs

![Hoskbrew Banner](public/logo.png)

Hoskbrew is a white-label physical game manufacturing service helping indie developers turn their digital games into real cartridges for NES, SNES, Game Boy, and Genesis.

## üöÄ Features

- **Professional Branding:** Custom Cthulhu-themed visual identity with "Bangers" and "Inter" typography.
- **Responsive Design:** Fully mobile-optimized layout built with Tailwind CSS.
- **Interactive UI:** Smooth Framer Motion animations for cards, buttons, and page transitions.
- **Quote Calculator:** Multi-step form for estimating manufacturing costs.
- **Platform Showcase:** Detailed specs for supported consoles (NES, SNES, GB, GBA, Genesis).
- **SEO Optimized:** Meta tags, sitemap, and robots.txt configured for indexing.

## üì¶ Getting Started

1. **Clone the repository:**

   ```bash
   git clone https://github.com/yourusername/hoskbrew-website.git
   cd hoskbrew-website
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Run the development server:**

   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
